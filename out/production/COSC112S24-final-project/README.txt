To compile, type the command:
javac *.java

To run, type:
java Main

To use the program:
The program is currently being run with hard-coded user data with a set name, age, sleep goal, bedtime, and wake time. When run, it should print out the user's bedtime, wake time, and the recommended bedtime depending on the user's data.
